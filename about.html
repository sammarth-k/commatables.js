About this Project

commatables.js is a parser that allows you to create tables with ease via the delimiters ",", ";" and "|" as well as files hosted on your domain or elsewhere. It simplifies your markup for large tables and if you use a file it will make it easier for you to edit your data directly in the file and see the changes automatically appear on your webpage. By using our CSS library, commastyles.css, your CommaTables will automatically become responsive. Furthermore, you will have access to many table themes and styles which are regularly updated.

This project was started by Sammarth Kumar and is licensed under a MIT License. The scripts and styles used on this website have been modified for demonstration purposes. The original code can be accessed via this GitHub repository.

This is an Open-Source project and you can modify your copy of the code however you want. If you have any suggestions or want to collaborate on new feautres/contribute to existing ones, you can email me here.
